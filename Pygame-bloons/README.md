
Opdracht
In deze onderwijsperiode gaan jullie een game maken die gelijkenissen heeft met het spel Bloons.
Figuur 1 – Bloons game poster
Dit is een game met heel veel objecten en een relatief eenvoudige gameplay. Het is echter
ontzettend leuk om te spelen. De Bloons TD 6 is grafisch ook erg fraai met hele leuke elementen. Ook
de geluidseffecten zijn heel leuk en effectief.
Onderstaand een voorbeeld van een level.
